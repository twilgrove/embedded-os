# 嵌入式操作系统

**本书由 [Claude Code](https://claude.com/claude-code) 创作。**

> 从晶体管到机器人系统，一本九卷 62 章、约 29 万字的嵌入式操作系统自学教材。面向有单片机 / RTOS / Linux 基础、想真正吃透内核与底层原理的工程师。

一本自包含的 HTML 教材，每一章一个网页，手机 / 电脑离线可读。讲法坚持「先讲为什么，再讲是什么」：先用生活类比引出问题，再一步步推导原理，最后落到源码、ARM 硬件和工程实践，尽量不突然抛术语。

## 适合谁

适合有 **单片机、实时操作系统 (RTOS)、Linux 系统开发** 基础的人——会 C、玩过 STM32 / FreeRTOS / RT-Thread、写过 Linux 应用或 ROS2，但对 **内核、ARM 架构、RTOS 内部实现** 还缺一层系统的底层认知。

## 内容（九卷 62 章）

| 卷 | 主题 | 章节 |
|----|------|------|
| 第一卷 | 计算机组成原理 | CPU / 指令 / 寄存器 / Stack / 中断 / Cache / MMU / 虚拟内存 |
| 第二卷 | ARM 架构 | Cortex-M/A、异常等级 EL0–EL3、GIC、Cache 一致性、TrustZone |
| 第三卷 | FreeRTOS 源码 | 调度器、PendSV 切换、Queue、信号量、Mutex 与优先级继承、Heap |
| 第四卷 | RT-Thread 源码 | 内核对象、IPC 与设备框架、内存 / FinSH / BSP |
| 第五卷 | Linux 内核 | task_struct、CFS、Buddy/Slab、VFS、Signal、Socket、Epoll、SoftIRQ、并发同步、RCU |
| 第六卷 | Linux 驱动 | 字符设备、总线-设备-驱动模型、设备树、GPIO、DMA、UART/I2C/SPI/CAN、mmap/ioctl、USB/PCIe |
| 第七卷 | 实时系统 | 实时性、延迟来源、RMS/EDF 调度、优先级反转、WCET、PREEMPT_RT |
| 第八卷 | RK3588 工程实践 | 架构总览、SMP、AMP、RemoteProc/RPMsg、共享内存与零拷贝、ROS2 |
| 第九卷 | 面试宝典 | 八大主题高频题精编 |

每章大致遵循同一套结构：**为什么需要 → 生活例子 → 定义 → 底层原理 → 数据结构 → 流程 → 源码分析 → 各 OS 实现对比 → ARM 硬件 → 工程应用 → 面试题 → 常见误区 → 总结 → 思维导图 → 课后练习**，配大量 ASCII 图解。

## 怎么读

- **电脑**：打开 [`index.html`](index.html)，点任意章节即可，章内有上一章 / 目录 / 下一章导航。
- **手机（推荐）**：直接用 **[`嵌入式OS教材-合订本.html`](嵌入式OS教材-合订本.html)** ——全书 62 章合成一个文件，只需下载这一个，章节间全部页内跳转，永远不会出现「文件找不到」，右下角有回顶按钮。
- 分章版在手机上如果点链接报 `FILE_NOT_FOUND`，是浏览器把单个文件复制进缓存目录、看不到同目录其它文件所致；用合订本即可彻底避开，或用 `file://` 完整路径 / 本地 HTTP 服务打开整个文件夹。
